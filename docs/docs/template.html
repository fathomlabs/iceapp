Installing and running IceApp

This is the IceApp documentation.

If you find a big or get stuck, please post feedback at the issue tracker.

IceApp is designed to be simple to use.

First, download the .dmg file (~280MB).

If you double-click that file, it will open up a new window.

Drag the ice cube icon into the applications folder icon to install IceApp.

Finally, open the Ice application from the Dock > Applications menu at the bottom of the screen.

An ice cube icon will appear in the taskbar. Clicking on it will show a dropdown menu that lets you launch a browser to connect to your ICE server.

Because this is a local installation it uses a self-generated security certificate. You'll need to accept the security exception in your browser on first visit.

If you've never run Ice before, you can log in with the username Administrator and password Administrator. Make sure you change the password once you're in!
